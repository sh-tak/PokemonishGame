
# Pokemonish Game   [![Badge License]][License]   ![Badge Status]


*実験ソフトウェア制作用のコード*

<br>

## 説明

ポケモンの対戦を模したゲーム <br>
最大128人がサーバーに登録可能 <br>
対戦して勝ったらモンスターが強くなります(ステータス強化)  

<br>
<br>

## 実行方法

### Windows

1. を起動

    ```shell
    server.bat
    ```
    
2. を起動

    ```shell
    client.bat
    ```

<br>

### Others

*コンパイルして実行*

- `server/Server`
- `client/GraphicalClient`
